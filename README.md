# Threshold (working name)

Short, guided practices for directly experiencing what contemplative traditions
point to, drawn from multiple traditions (Zen, Advaita, Sufism, Taoism, and
others) — not more explanation of what they mean.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js (App Router) + TypeScript + Tailwind v4, deployed on Vercel.
